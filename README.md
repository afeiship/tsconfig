# tsconfig
> About [Shared TypeScript](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html) config for my projects.

## installation
```shell
npm i @jswork/tsconfig
```

## usage
```json
{
  "extends": "@jswork/tsconfig",
  "compilerOptions": {
    "outDir": "dist",
    "target": "ES2021"
  }
}
```
